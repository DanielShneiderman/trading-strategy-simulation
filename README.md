# Trading Strategy Monte Carlo Simulation

This project simulates a trading strategy using random outcomes to analyze performance, risk, and drawdowns.

## Overview
The simulation runs multiple trading scenarios with a fixed risk/reward profile (win = 10, loss = -8) to evaluate how randomness impacts trading results over time.

## Metrics analyzed
- Expected profit
- Drawdowns (risk)
- Profit distribution
- Drawdown distribution
- Risk-adjusted return

## Key insights
- Even profitable strategies experience drawdowns
- Outcomes vary significantly due to randomness
- Risk management is critical for long-term performance

## Tools used
- Python
- Matplotlib
- Statistics

## How to run
Open the notebook in Google Colab or Jupyter and run all cells.
